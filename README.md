### ⚡ How to use

1. Pick a prompt from `/prompts/hu/` or `/prompts/en/`
2. Fill in the `[VARIABLES]`
3. Activate modules with `+` / deactivate with `-`
4. Paste into any LLM — Claude, Gemini, ChatGPT, Gemma
